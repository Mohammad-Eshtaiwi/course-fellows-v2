# Quick SEO Checklist ✅

## Immediate Actions (Before Launch)

- [ ] **Set `NEXT_PUBLIC_BASE_URL` in your environment**
  ```bash
  NEXT_PUBLIC_BASE_URL=https://yourdomain.com
  ```

- [ ] **Deploy your app** to production (Vercel, Netlify, etc.)

- [ ] **Verify sitemap works**: Visit `yourdomain.com/sitemap.xml`

- [ ] **Verify robots.txt works**: Visit `yourdomain.com/robots.txt`

## Google Search Console Setup (Day 1-3)

- [ ] Go to [Google Search Console](https://search.google.com/search-console)
- [ ] Add your property (domain)
- [ ] Verify ownership (add verification code to `app/layout.tsx`)
- [ ] Submit sitemap: `sitemap.xml`
- [ ] Request indexing for homepage

## Week 1-2

- [ ] Check Search Console for crawl errors
- [ ] Verify pages are being indexed (use URL Inspection tool)
- [ ] Add Google Analytics (optional but recommended)
- [ ] Share on social media to get initial traffic

## Week 3-4

- [ ] Monitor impressions and clicks in Search Console
- [ ] Fix any coverage issues reported
- [ ] Add more content/pages if needed
- [ ] Optimize meta descriptions based on CTR

## Ongoing

- [ ] Post regular content/updates
- [ ] Build backlinks (social media, forums, guest posts)
- [ ] Monitor Core Web Vitals
- [ ] Update sitemap as you add new courses/pages

---

## Quick Tests

**Is my site indexed?**
```
Search Google for: site:yourdomain.com
```

**Are my pages showing up?**
```
Search Google for: site:yourdomain.com/courses
```

**Check page speed:**
- Visit: https://pagespeed.web.dev/
- Enter your URL
- Aim for 90+ score
